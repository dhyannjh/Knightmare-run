# Knightmare-run
(pun intended with the name)
This project is a Godot-based 2D platformer where you control a knight and progress through hazards using checkpoints. It started as a tutorial project and was expanded with additional features and systems. Big thanks to the original tutorial creator, Brackeys, for the base and inspiration.

#### Link: https://www.youtube.com/watch?v=e1zJS31tr88

## Gameplay 
🪙 Collect coins
⚔️ Avoid or defeat enemies
🚩 Reach checkpoints to save progress
💀 Die and respawn using a custom system
🏁 Reach the end to win

### Controls 
WASD or Arrow Keys allong with space for jump
