# Knightmare-run
(pun intended with the name) This project is a Godot-based 2D platformer where you control a knight and progress through hazards using checkpoints. It started as a tutorial project and was expanded with additional features and systems. Big thanks to the original tutorial creator, Brackeys, for the base and inspiration.

#### Link: https://www.youtube.com/watch?v=e1zJS31tr88

## Gameplay 
1. 🪙 Collect coins
2. ⚔️ Avoid or defeat enemies
3. 🚩 Reach checkpoints to save progress
3. 💀 Die and respawn using a custom system
4. 🏁 Reach the end to win

[Gmaeplay](SCREENSHOTS/Gamplay.gif)

### Controls 
WASD or Arrow Keys allong with space for jump

## Built With

1. Godot
2. GDscript
